# kuis-ular-tangga
Kuis ular tangga adalah kombinasi seru dari kuis interaktif dan permainan klasik ular tangga.Lempar dadu dan jawab pertanyaan dengan benar untuk melangkah maju.Kamu memiliki 100 nyawa.Jika kamu salah menjawab 5 nyawamu akan hilang.Kotak hijau (tangga),Kotak merah (ular),Kotak biru (memulihkan nyawa).Capai kotak untuk menang
